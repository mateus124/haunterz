Script Hub

Nessa área você encontrará diversos scripts aprovados pela equipe da Haunter Z, tanto em parcerias como em utilizações diárias. Que são julgados como os melhore da atualidade, e sempre serão atualizados constantemente, sem que você se preucupe com versão do script/game que irá utilizar.

Como usar

Para utilizar o script hub da forma correta, basta descer até onde se encontram as postagens de script, serão encontradas imagens e a descrição de cada script, ao clicar em algum deles, o campo de texto abaixo será alterado com o código do script, após isso, basta clicar no botão "Copiar" e utilizar seu script no executor da Haunter Z!
